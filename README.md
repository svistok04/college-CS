# College-CS

Repository containing my Computer Science coursework, organized by semester.

---

## Structure

### Semester 1
- `programming/` — C fundamentals (Programming 1)

### Semester 2
- `programming/` — C++ fundamentals, STL and OOP (Programming 2)
- `discretemath/` — discrete mathematics projects (Python)
- `verilog/` — basic hardware description (Verilog)
- `R/` — statistics

### Semester 3
- `Algorithms 2/` — algorithms and data structures
- `Operating Systems/` — OS concepts and labs (Linux and Windows)
- `Numerical Analysis/` — numerical methods (MATLAB)
- `Python/` — Python fundamentals
- `ASK/` — low-level programming in assembly

### Semester 4
- `graphics/` — computer graphics (Blender, ray tracing in C++, Unity)
- `data-analysis/` — data analysis and signal processing basics (Python, Jupyter Notebook)
- `signal-processing/` — digital communication systems simulation (modulation, channel noise, spectral analysis, BER etc.)
- `databases/` — SQL, XML, data modeling
- `software-eng/` — team project (collaborative repository with multiple contributors)

### Semester 5
- `avr-bare-metal/` — low-level embedded programming (AVR)
- `AI/` — search algorithms, heuristics, and basic machine learning (Java, Python)
- `HMI/` — human–machine interaction (gesture recognition, sensors, voice etc.; Python)

### Semester 6
- `compute-acceleration/` — OpenCL and parallel computing
- `embedded/` — STM32 and embedded systems
- `modeling-and-simulation/`— dynamic systems modeling and simulation (MATLAB/Simulink)
- `multimedia-systems/` — image and audio processing (scaling, quantization, compression, resampling; Python)
- `software-design/` — object-oriented design, design patterns, and refactoring (Java)
- `system-identification/` — system identification and parameter estimation from data (MATLAB/Simulink)

---

- The `software-eng` directory is a team project and includes contributions from multiple collaborators.
